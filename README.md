# nico-time

MT4用インジケーター。時刻ライン・キリバン・前日高値安値・
日足/週足ピボットを1本のチャートに一括表示します。

## ファイル

- `nico-time.mq4` — インジケーター本体
- `nico-time.set` — プリセット設定ファイル

## インストール方法

1. `nico-time.mq4` をダウンロード
2. MT4メニュー →「ファイル」→「データフォルダを開く」
3. MQL4 → Indicators フォルダに貼り付け
4. MT4を再起動（またはF7でコンパイル）
5. ナビゲーター →「nico-time」をチャートにドラッグ
